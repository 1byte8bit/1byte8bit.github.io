Element UI之Cascader 级联选择器多选操作

Dec 24, 2019

1、概要

1.1、级联选择器，树型展示

Element UI的官网链接：https://element.eleme.cn/#/zh-CN/component/cascader
使用环境：
"element-ui": "2.9.0",
需要依赖为2.9.x及以上，配合
"vue": "2.5.22" 以上。
如果版本不匹配，多选操作是显示不出来的。

1.2、需求描述

数据库中存在两张表，user表和org表，user表中有字段org_id，记录这个人所属的组织id，org表中有org_id，org_name等字段。
表结构简介如下
user:

id name org_id …
1 admin 10000000 …

org

org_id org_name parent_org_id
00000000 刑部 -1
10000000 六扇门 00000000
10000001 总管 10000000
10000002 办公室 10000000
10000003 兵器管理 10000000
10000003 财政 10000000
10000003 保洁 10000000
  • 目前每个人只有一个所属组织
  • 管理员，部门负责人在进行组织查询的时候，只能有本部门及其子部门的展示权限
  • 虽然不属于部门B，但是想看到部门B的信息，单选满足不了，靠两个表维护不了多选的信息。

基于以上来自产品顾问天马星空的想法，对现有的情况进行改造。
